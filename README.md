✊📄✂️ CODSOFT Task 4 - Rock Paper Scissors Game

📌 Project Overview

This project is a modern Rock Paper Scissors Game developed using Python and CustomTkinter as part of the CODSOFT Python Programming Internship.

The application allows users to play against the computer in a visually appealing desktop interface. It features real-time score tracking, game result updates, a reset option, and an interactive layout for an engaging user experience.

---

✨ Features

- 🎮 Play Rock, Paper, Scissors against the computer
- 🤖 Random computer choice generation
- 🏆 Real-time score tracking
- 📊 Displays both player and computer scores
- 🔄 Reset game functionality
- 📋 Built-in game rules
- 🌙 Modern dark-themed interface using CustomTkinter
- 🖥️ Responsive and user-friendly GUI

---

🛠️ Technologies Used

- Python 3
- CustomTkinter
- Random Module
